Chart-First

ทุกรายงานต้องมีกราฟอย่างน้อย 1 อัน — Line, Bar, Doughnut, Scatter, Sparkline เลือกตามข้อมูล

Dark/Light Theme

ตรวจสอบ system preference อัตโนมัติ สลับโหมดได้ด้วยคลิกเดียว CSS variables ทั้งหมด

✂️

Caveman-Style

ไม่มีข้อความฟุ่มเฟือย — "Revenue" ไม่ใช่ "Total Revenue for Q4 2026" ข้อมูลแน่นทุกพิกเซล

📄

ไฟล์เดียวจบ

HTML ไฟล์เดียว รวม CSS + Chart.js ผ่าน CDN ไม่ต้อง build, ไม่ต้อง dependencies, แชร์ได้ทันที

🖨️

พิมพ์สวย

@media print styles ในตัว — พิมพ์ออกกระดาษก็สวย ไม่ต้องปรับแต่งเพิ่ม
